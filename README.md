# tsfetch
Command-line system information tool written in TypeScript.

## install
```
npm i tsfetch-re
```

```
$ tsfetch
MBRjun@MBR-PC
-------------
OS: Windows 11 Pro for Workstations x64
Kernel: 10.0.22631
Uptime: 1 day, 3 hours
CPU: 13th Gen Intel Core i5-13600KF (20) @ 3.49GHz
Memory: 17.6 GiB / 31.8 GiB
```
